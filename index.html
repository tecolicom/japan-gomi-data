japan-gomi-data

日本の自治体の「ごみの日」を、機械可読なオープンデータに。

GitHub 解説記事 データの説明

自治体が公式に発表しているごみ収集日程を、共通の形式 (収集規則+例外日) に整理して CC BY 4.0 で公開しています。 現在 — を収録。各自治体の公式資料と照合した検証記録つきです。

カレンダーで購読する

「カレンダーに追加」でお使いのカレンダーアプリ (iPhone・Google カレンダー等) に収集日が終日予定として入ります。 自分の地区がどのコースかは、コース名の下の対象地区で確認してください。

データ

  • 収集日程データ (YAML) — 規則+例外日の構造化データ。スキーマつき
  • コース一覧 (CSV) — 全コースの自治体コード・対象地区・購読 URL
  • 調査台帳 (CSV) — 東京・埼玉 125 自治体の収集日データ公開状況。解説記事

収集日程は各自治体の公式発表に基づきますが、機械変換のため誤りが残り得ます。実際のごみ出しは自治体の公式情報も併せてご確認ください。 誤りに気づいた場合は GitHub Issues へどうぞ。
