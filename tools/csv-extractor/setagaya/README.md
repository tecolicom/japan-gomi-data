# setagaya (世田谷区) CSV 抽出パイプライン

区オープンデータ CSV「資源・ごみ収集曜日一覧」(CC BY 4.0) を一次ソースに、
`municipalities/tokyo/setagaya/2026/course-*.yaml` を生成する。

```
node fetch.mjs        # CSV + 416.html + 27859.html + カレンダーPDF 41枚 + all.pdf → cache/
node --test           # パーサ単体テスト
node build.mjs        # cache/ → course-*.yaml (118 町丁目 → 37 コース)
node verify.mjs --month 2026-11 --month 2026-12   # 期待日程を出力 (PDF 目視照合用)
node verify.mjs --render 8                        # 全 37 PDF の該当ページを PNG 化
```

`fetch.mjs` は直接実行したときだけ取得する (`build.mjs` は URL 定数だけを import する)。
再取得は `node fetch.mjs --force`。

## ソースと照合の構造

- **一次ソース**: オープンデータ CSV (Shift-JIS)。列は
  `50音, 町名, 丁目, 資源(週1回), 可燃ごみ(週2回), 不燃ごみ(月2回), ペットボトル(月2回), 管轄清掃事務所` の 8 列・118 行。
- **照合ソース 1 (機械)**: 同じ配布ページ `416.html` の HTML 表。同内容だが表記が異なる
  (CSV「水曜日・土曜日」「2回目・4回目の月曜日」/ HTML「水・土曜日」「2・4回目の月曜日」)。
  `build.mjs` が正規化して**全 118 行・全列を突合**し、1 つでも違えば中断する。
- **照合ソース 2 (構造)**: カレンダー配布ページ `27859.html` の町丁目 → 対象地区 PDF 対応。
  CSV を日程シグネチャで畳んだ 37 群が、区自身の「対象地区」1〜37 のグループ分けと
  **完全一致すること**を `build.mjs` が要求する。コース番号はこの対象地区番号をそのまま使う
  (course "12" = `no12.pdf` = 対象地区⑫)。
- **照合ソース 3 (日付レベル)**: 町丁目別カレンダー PDF 本体。**全ページが画像**で
  テキスト層は 8 文字 (改ページのみ) しかないため機械抽出できない。`verify.mjs` で
  期待日程の出力と PDF ページの PNG 化を行い、**目視で**照合する (2026-07-20 に全 37 地区実施)。

## PDF の罠

- **画像 PDF**。`pdftotext` は 8 文字しか返さない。一覧 PDF (`syusyubi.pdf`) も同様に 27 文字で
  実質画像。機械照合には使えないので `pdftoppm` で PNG 化して読む。
- **「収集はありません」は休止日とは限らない**。カレンダーは
  「その曜日には当たるが n 回目に該当しない」空欄にもこの注記を入れる (例: 5 回目の水曜)。
  **黄色地**で塗られたセルだけが本当の休止日。2026年度で黄色なのは **12/31 のみ**で、
  12/30 は通常収集 (対象地区6・7・8等は 12/30 水曜に可燃ごみを収集する)。
- カレンダーの収録範囲は **2025年12月〜2026年12月** で、会計年度 (4月〜翌3月) とずれる。
  2027年1〜3月は範囲外のため日付レベルの照合ができない。
- 祝日は通常収集 (昭和の日・海の日・山の日・文化の日・勤労感謝の日で収集チップを確認)。

## ページ構成 (令和8年版 no<N>.pdf, 全 8 ページ)

| p1 | p2 | p3 | p4 | p5 | p6 | p7 | p8 |
|---|---|---|---|---|---|---|---|
| 表紙 (対象地区・町丁目) | 令和7年12月 | 1・2月 | 3・4月 | 5・6月 | 7・8月 | 9・10月 | 11・12月 |

`all.pdf` は全地区共通の解説部 (P9-23)。分別区分は P12-13 (資源=古紙+ガラスびん+缶)。

## 対象外

- 対象地区 **no38〜no41** は大規模集合住宅の個別日程 (防衛省池尻宿舎・東京テラス・希望ヶ丘団地
  ・アクティ三軒茶屋 等 12 件)。町丁目 CSV に現れないため収録していない。
- 古着・古布と一部の古紙は集団回収 (区収集外)、粗大ごみは申込制のため対象外。

## 来年度更新

1. 区は**毎年11月頃**に翌年版カレンダー (12月〜翌12月) を配布し、
   **収集曜日そのものが改定される可能性がある** (27859.html に明記)。11月に必ず再確認する。
2. `build.mjs` の `YEAR`・`fiscal_year_ja`・`YEAR_END` を更新
   (新カレンダーで年末年始の黄色セルを確認し、2027年始 1/1〜1/3 の休止を反映する)。
3. `fetch.mjs --force` → `node --test` → `build.mjs` → `verify.mjs` を再実行。
   CSV と HTML 表の突合、または対象地区グループの一致が落ちたら日程改定を疑う。
